# Generative NFT Art

A comprehensive toolkit for creating unique algorithmic artwork suitable for NFT deployment on platforms like HIGHLIGHT.XYZ.

## 🎨 Overview

This project provides a robust framework for generating unique digital art using various algorithmic approaches including:

- **Perlin Noise Flow Fields** - Organic, flowing patterns driven by mathematical noise
- **Fractal Algorithms** - Self-similar patterns with infinite complexity
- **Cellular Automata** - Emergent behaviors from simple rules
- **Geometric Patterns** - Mathematical precision meets artistic expression
- **Particle Systems** - Dynamic simulations creating natural movements

## ✨ Features

- **Deterministic Generation** - Same seed always produces identical artwork
- **Multiple Algorithms** - Extensible architecture for adding new art generators
- **Color Theory Integration** - Sophisticated palette generation and color harmony
- **High-Quality Output** - Support for various formats (PNG, SVG, Canvas)
- **HIGHLIGHT.XYZ Ready** - Optimized for generative NFT platform deployment
- **TypeScript Support** - Type-safe development with comprehensive interfaces
- **Testing Framework** - Comprehensive testing including visual regression tests

## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/nibertinvestments/GenerativeNFTART.git
cd GenerativeNFTART

# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test
```

### Generate Your First Artwork

```bash
# Generate a single piece
npm run generate single "my-unique-seed"

# Generate a batch of 10 pieces
npm run generate batch 10

# List available algorithms
npm run generate list
```

### Using the Library

```javascript
import { generate, generateWithParameters } from './src/index.js';

// Simple generation
const canvas = generate('unique-seed-123');

// Generation with custom parameters
const result = generateWithParameters('unique-seed-123', 'Perlin Flow Field', {
  particleCount: 3000,
  noiseScale: 0.008,
  palette: 'sunset'
});

// Access the canvas and metadata
console.log(result.features);
document.body.appendChild(result.canvas);
```

## 🎯 Available Algorithms

### Perlin Flow Field
Particles follow vector fields generated by Perlin noise, creating organic, flowing patterns.

**Parameters:**
- Particle Count (500-5000)
- Noise Scale (0.001-0.02)
- Flow Strength (0.5-5.0)
- Trail Length (5-50)
- Color Palette (predefined sets)

**Features Generated:**
- Flow Pattern (Turbulent/Organic/Chaotic)
- Energy Level (Calm/Dynamic/Intense)
- Complexity Score (1-10)

## 🎨 Color Palettes

The system includes carefully curated color palettes:

- **Ocean** - Deep blues and aqua tones
- **Sunset** - Warm oranges, reds, and purples
- **Earth** - Natural browns and tans
- **Cool** - Blues and purples
- **Warm** - Reds, oranges, and yellows

Additional palettes can be generated procedurally using color theory algorithms.

## 🛠 Development

### Adding New Algorithms

1. Create a new file in `src/algorithms/[category]/`
2. Extend the `Generator` base class
3. Implement required methods: `generateFeatures()` and `render()`
4. Register the generator in `src/index.ts`

Example:
```typescript
import { Generator, GenerationContext } from '../../core/Generator.js';

export class MyCustomGenerator extends Generator {
  constructor() {
    super('My Algorithm', 'Description of what it does');
    // Define parameters...
  }

  protected generateFeatures(context: GenerationContext) {
    // Return feature object
  }

  protected render(context: GenerationContext) {
    // Implement rendering logic
  }
}
```
